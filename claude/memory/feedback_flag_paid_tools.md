Always mention upfront when a recommended tool, API, or service requires a paid subscription or has per-use costs before suggesting it.

**Why:** User didn't want Brave Search API after learning it was a subscription — would have preferred to know before it was wired in.

**How to apply:** When recommending any external API, MCP server, SaaS, or cloud service, state the pricing model (free tier, paid, usage-based) in the same sentence as the recommendation. Don't make the user ask.
